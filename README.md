# Joe Tracker

**Деплой на GitHub Pages:**

```bash
# 1. npm install
# 2. Открой public/generate-icons.html → сохрани 3 PNG в public/icons/
# 3. git init && git add . && git commit -m "init"
# 4. gh repo create Joe-Tracker- --public --push --source=.
# 5. GitHub → Settings → Pages → Source: gh-pages branch
```

Автодеплой: push в `main` → GitHub Action → https://USERNAME.github.io/Joe-Tracker-/

**Dev:**
```bash
npm install && npm run dev
```
